# User Login Register System

This is a small full-stack project for learning frontend and backend development.

The goal is to understand how the frontend interacts with the backend.

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript, Vue.js (CDN)
- **Backend:** Python (fastapi+uvicorn+sqlalchemy)
- **Authentication:** JWT
- **Database:** Postgresql

## Key Learning Points

#### Backend Development

- RESTful API design.
- User schemas and models design.
- Use sqlalchemy orm interact with database.
- User argon2 hashing password.

#### Frontend Development

- Using Vue.js (CDN) for reactive user interfaces.
- Making asynchronous API calls to the backend.
- Managing user input and application state.

#### JWT

- Understanding stateless authentication mechanisms.
- Generating and verifying tokens for secure access.

#### Database Management

- Designing user data schemas.
- Securely storing user credentials (e.g., password hashing).
- Performing database CRUD operations.
